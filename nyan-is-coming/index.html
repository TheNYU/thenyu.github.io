Nyan · Nyan is coming!

Open source game engine for RPG Maker

Blog · 2017-01-15

The development of Nyan is very active since 2 months. And this post is the first of this blog. Here is what I plan for Nyan for the future, concerning the release.

Warning: it can change at any moment without any explanation

  • January 2017: presentation of Nyan in forums related to RPG Maker (french and english forums)
  • Late February - early March 2017: private alpha
  • Late March - April 2017: first public alpha with only base functionalities for debugging purpose with poll for adding missing functionalities
  • Summer 2017: first public beta for debugging purpose
  • Late 2017: first stable release with all functionalities available

However, you can now wait for Nyan!
